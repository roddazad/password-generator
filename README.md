# password-generator
Generates random password


## Overview
This is a simple and professional Password Generator web application. The application allows users to generate secure passwords based on selected criteria including length, inclusion of uppercase letters, numbers, and special characters.

## Features
- Specify the length of the password (8 to 128 characters).
- Option to include special characters.
- Option to include uppercase letters.
- Option to include numbers.
- Instant generation and display of the password.

## Technologies Used
- HTML
- CSS
- JavaScript

## Getting Started
Follow these instructions to get a copy of the project up and running on your local machine.

### Prerequisites
modern web browser installed on your machine.


### Usage
1. Open `index.html` in your browser.
2. Specify the desired password length.
3. Select the options for including special characters, uppercase letters, and numbers.
4. Click the "Generate Password" button.
5. The generated password will be displayed in the result section.

## Project Structure
- `index.html`: The main HTML file containing the structure of the web page.
- `style.css`: The CSS file for styling the web page.
- `script.js`: The JavaScript file containing the logic for generating the password.
